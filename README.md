# VIGIL

Low-cost continuous patient monitoring headband.

## Features

- Heart rate via BCG (LIS3DH accelerometer)
- Temperature via TMP117
- Respiratory rate via MEMS microphone (ICS-43434)
- Pulse detection via PVDF piezoelectric sensors
- BLE connectivity via nRF52840

## Specs

| Spec | Value |
|------|-------|
| Dimensions | 185mm × 35mm |
| PCB | 2-layer Flex |
| Thickness | 0.11mm |
